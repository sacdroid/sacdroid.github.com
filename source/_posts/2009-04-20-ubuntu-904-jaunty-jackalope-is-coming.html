With the final release of Ubuntu 9.04 “Jaunty Jackalope” only days away (scheduled for April 23, 2009), I already installed Release Candidate.

So, what’s new?
  • GNOME 2.26.1 desktop environment (KDE 4.2.2 for Kubuntu)
  • Kernel packages were updated to version 2.6.28-11.14, based on Linux kernel 2.6.28.9
  • Faster boot times (18 seconds!!!)
  • ext4 filesystem support
  • Wacom tablet support
  • Netbook supp
  • KDE bluetooth fixes!!
